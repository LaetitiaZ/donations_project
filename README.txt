You'll find here the code for a donation management application. 

The goal of the association «Mobilier_Pour_Tous» is to help ederly people who are in
difficult situation. The association can provide furniture or other material goods to the
ones in need. Therefore, the association is often solicited to receive and give donations.
In order to manage the flow of donations, an application is programmed using JAVA. 

When the code is first being executed, some .txt files are generated.
These correspond to the initial situation (current members and donations).

The services provided by the application differ according to the user's status (members, president, treasurer)
It keeps track of all transactions in files and make some statistics out of it.
